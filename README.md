# 2026 SF Good Neighbor Week Map

An interactive, multi-layered dashboard of San Francisco's community infrastructure, designed to foster hyper-local connection during the **2026 SF Good Neighbor Week**. 

Inspired by the work of Stephen Braitsch, this web application neighborhood boundaries and layers of civic resources loaded dynamically from (for now) file-based CSV tables.

This site should remain front-end only, and prioritizes a beaufiful experience for community engagement.
